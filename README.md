## Sobre

Portfolio pessoal desenvolvido com design moderno e animações 3D interativas. Projetado para展示 minhas habilidades como desenvolvedor Fullstack com 3+ anos de experiência.

🔗 **Acesse o site:** [portfolio-virid-mu-84kv6g7goh.vercel.app](https://portfolio-virid-mu-84kv6g7goh.vercel.app)

## Funcionalidades

- Background 3D animado com Three.js
- Cursor personalizado com efeito de seguimento
- Animações de scroll suaves
- Efeito de digitação na hero section
- Modal detalhado para cada projeto
- Mockups realistas dos projetos
- Totalmente responsivo (mobile-first)
- Menu hamburger para mobile
- Formulário de contato
- Easter egg: Konami Code

## Como Executar

```bash
# Clone o repositório
git clone https://github.com/Diasprogramador/Portfolio-caio.git

# Entre na pasta
cd Portfolio-caio

# Abra o index.html no navegador
# ou use um servidor local
npx serve .
```
